# UC07 Accès & Conformité

## Expression de besoin fonctionnelle
- **Objectifs** : gérer droits d’accès, audits, demandes et revues de conformité.
- **Acteurs** : IAM, compliance, managers, utilisateurs.
- **Flux clés** : demande accès → approbation → provisioning → revue périodique → audit.
- **Exceptions** : accès d’urgence temporaires; révocation immédiate en cas d’incident.
- **Critères d’acceptation** : RBAC, journaux d’audit, revues automatiques, traçabilité complète.

## Analyse ROI
- **Gains** : réduction des risques de non‑conformité, baisse du temps d’audit.
- **Coûts** : intégrations IAM, paramétrage règles, formation.
- **KPI cibles** : délai moyen d’octroi d’accès, % revues réalisées, nombre de findings d’audit.
